## maemo-kodi-remote

A simple and easy-to-use Kodi remote.

Made for devices with a physical keyboard.

### Features

- Multiple Kodi instances in one app
- Auto-discovery on LAN
- Media thumbnail
- Navigation
- Volume control
- Mute/unmute
- Sending keystrokes
- Choose audio stream
- Choose subtitle
